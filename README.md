# IPNC
Indo Phone Number Checker (IPNC). Tools for checking your Indonesia phone number information.


## Installation & Run the Program

So easy.

```bash
pkg install python3
pkg install requests
pkg install toilet
pkg install git
git clone https://github.com/danrfq/IPNC
cd IPNC
python3 IPNC.py
```
